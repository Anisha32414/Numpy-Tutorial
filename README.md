# 📊 NumPy Tutorial

This repository provides a beginner-friendly guide to **NumPy**, one of the most widely used Python libraries for numerical computing and data analysis.

Each topic includes **concise notes** explaining the concepts along with **practical code examples** to help build a strong understanding of NumPy fundamentals.

---

## 📚 Topics Covered

* NumPy Arrays
* Array Creation
* Indexing & Slicing
* Reshaping Arrays
* Mathematical Operations
* Broadcasting
* Aggregation Functions
* Random Module
* Linear Algebra
* Array Manipulation
* Useful NumPy Functions

---

## 🎯 Repository Objectives

* Learn NumPy from basics to intermediate level.
* Understand concepts through notes and hands-on examples.
* Build a strong foundation for Data Science, Machine Learning, and AI.
